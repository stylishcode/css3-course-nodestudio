ClearFix é uma técnica usada para reparar problemas causados pelo uso do float. Vale lembrar que você só precisa dessa técnica se for construir layouts com floats, se o layout for flexbox, não é necessário e com frameworks também não, pois nem teriamos acesso aos códigos css, que são abstraidos pelo framework

Contextualizando o problema:

- Se quisermos mover a div box2 para direita, usamos o float right, porém isso vai quebrar o layout (div container perde o fundo na mesma linha da box2 e talvez outras coisas) e mesmo que eu mova a div box1 para a esquerda com o float left tentativa de resolver , o problema também acontece.

Formas de resolver:

- Adicionando display table na div pai, isso resolveria o problema, mas desenvolvedores relataram problemas com o uso do display table, então não recomendado

- Adicionar float left na div pai, resolveria o problema, mas só funciona se a div tiver um tamanho fixo ou 100%, então só use com elementos de tamanho fixo, só que, se precisarmos centralizar a div com um margin 0 auto, o float fica inutil, e o problema volta, então não recomendado também

- Criar uma div class clearfix após o final do layout e na propriedade css aplicar um clear both nessa div, porém era muita poluição ficar criando clear fixes para lá e para cá nos containers para resolver esse problema nos layouts, era usado antigamente.

Agora com os pseudo elementos, tudo fica mais fácil, resolução no código css
